## 1.6 池塘夜降彩色雨(1)

做一个动画：五颜六色的雨点落入池塘，入水有声音，水面有涟漪。

图形界面C++库：EsayX/MFC。

基于MFC库：[基于VC++的MFC类库实现的池塘夜降彩色雨程序](https://www.writebug.com/git/Momenter/Color_rain)

基于EsayX库：[colorful_rain：池塘夜降彩色雨](https://github.com/misterchaos/colorful_rain)

## 2.4 马踏棋盘(2)

要求程序非递归。
